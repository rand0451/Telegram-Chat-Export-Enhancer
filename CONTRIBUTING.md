# Contributing

Thanks for your interest in contributing to Telegram Chat Export Enhancer! Contributions are welcome â€” issues, feature requests, and pull requests.

Please follow these guidelines:

- Open an issue first for major features or breaking changes.
- For bug fixes or small improvements, open a pull request targeting the `main` branch.
- Keep PRs small and focused.
- Add comments and brief tests (if applicable).
- Update `CHANGELOG.md` with a short summary of changes in the PR description.
- Follow the existing code style: ES6, modular functions, readable variable names.

How to run & test locally
1. Edit `telegram-chat-enhancer.user.js` in your favorite editor.
2. Save locally and open your userscript manager (Tampermonkey) to install the local file.
3. Open a Telegram HTML export file and test interactions.
4. Use the browser console to inspect errors.

Reporting security issues
- If you discover a security vulnerability, please open an issue and mark it as security. Do not post exploit details publicly.
